├── dist/

├── docs/                         # Документация
├── presentations/                # Презентации

Запуск из исходного кода

Основной способ развёртывания для Windows и Linux

1

Установите Python 3.8+

Скачайте с python.org и установите, отметив «Add Python to PATH» 

python --version
pip --version
2

Перейдите в директорию проекта

cd task-chat/app
3

Установите зависимости

pip install -r ../requirements.txt

Основные зависимости: Flask, Flask-Login, Flask-SQLAlchemy, Werkzeug, waitress (Windows) / gunicorn (Linux)

4

Запустите сервер

python run.py

Сервер запустится на http://localhost:5000

После первого запуска

В директории проекта создастся папка data/ с базой данных, логами и загруженными файлами.

Конфигурация

Настройка через переменные окружения

Переменная По умолчанию Описание
SECRET_KEY Встроенный Ключ сессии
DATABASE_URL sqlite:///data/instance/taskchat.db Строка подключения к БД
PORT 5000 Порт сервера
HOST 0.0.0.0 Хост для bind
FLASK_CONFIG development development / production
LOG_LEVEL INFO Уровень логирования

Установка переменных (Windows) 

# Временная установка (до перезапуска консоли)
set PORT=8080
set HOST=127.0.0.1
set LOG_LEVEL=DEBUG

# Постоянная установка — через «Переменные среды» в свойствах системы

Резервное копирование

Простое резервирование данных

База данных

Путь: data/instance/taskchat.db

Частота: Ежедневно

Файлы

Путь: data/uploads/

Частота: Ежедневно

Логи

Путь: data/logs/

Частота: Еженедельно (опционально) 

# Пример скрипта резервного копирования (Windows)
@echo off
set BACKUP_DIR=E:\Backups\TaskChat
set DATA_DIR=task-chat\data
set DATE=%DATE:~-4,4%%DATE:~-7,2%%DATE:~-10,2%

echo Creating backup: %DATE%
mkdir %BACKUP_DIR%

:: Резервное копирование БД
xcopy %DATA_DIR%\instance %BACKUP_DIR%\instance_%DATE% /E /I /Y

:: Резервное копирование файлов
xcopy %DATA_DIR%\uploads %BACKUP_DIR%\uploads_%DATE% /E /I /Y

echo Backup completed: %DATE%

Важно

Перед обновлением версии обязательно сделайте резервную копию директории data/
