Структура каталога: 
Task-Chat-Server-v0.3.3/
├── Task-Chat-Server.exe    # Портативный сервер (Windows)
├── run_server.sh           # Скрипт запуска (Linux)
├── app/                    # Исходный код
├── data/                   # ДАННЫЕ (БД, логи, файлы)
│   ├── instance/           # База данных
│   ├── logs/               # Логи
│   └── uploads/            # Загруженные файлы
└── venv/                   # Виртуальное окружение

Установка на Windows

1

Скопируйте каталог

Скопируйте Task-Chat-Server-v0.3.3\ в нужное место (например, C:\Servers\Task-Chat\)

2

Запустите сервер

Дважды кликните на Task-Chat-Server.exe или RUN_GUI.bat

3

Откройте браузер

Перейдите на http://192.168.x.x:5000

4

Войдите в систему

Логин: admin / Пароль: admin123

Готово!

Сервер запущен и готов к работе. GUI интерфейс управления откроется автоматически.

Установка на Linux

1

Скопируйте каталог

Скопируйте Task-Chat-Server-v0.3.3/ в нужное место

2

Дайте права на выполнение

chmod +x run_server.sh

3

Запустите сервер

./run_server.sh

4

Войдите в систему

Логин: admin / Пароль: admin123
