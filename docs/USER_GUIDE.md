## Начало работы

### Что такое Task-Chat?

Task-Chat — это система для управления задачами на медицинские исследования (CT, MR, PET) и коммуникации между врачами и физиками.

### Роли пользователей

| Роль | Описание |
|------|----------|
| **Врач** | Создаёт задачи на исследования, общается с физиками |
| **Физик** | Обрабатывает задачи, меняет статус, общается с врачами |
| **Администратор** | Управляет пользователями и всеми задачами |

## Вход в систему

### Первый вход

1. Откройте браузер (Chrome, Firefox, Edge)
2. Введите адрес: `http://localhost:5000`
3. Введите учётные данные:
   - **Логин:** `admin`
   - **Пароль:** `admin123`
4. Нажмите кнопку **«Войти»**

### Смена пароля

1. Администратор может сменить пароль любому пользователю в разделе **Админ → Пользователи**
2. Нажмите на иконку ключа рядом с пользователем
3. Введите новый пароль
4. Нажмите **«Изменить пароль»**

### Выход из системы

1. Нажмите на иконку **«Выход»** в верхней панели
2. Или нажмите на иконку выхода в мобильном меню

## Для врачей

### Создание задачи

**Только врачи могут создавать задачи.**

1. На главной странице нажмите кнопку **«Создать задачу»**
2. Заполните форму:

**Обязательные поля:**
- **Тип исследования:** выберите из списка
  - `Импорт CT` — компьютерная томография
  - `Импорт CT-диагностики` — диагностическая КТ
  - `Импорт MR` — магнитно-резонансная томография
  - `Импорт PET` — позитронно-эмиссионная томография
- **Карта пациента:** номер карты пациента

**Дополнительные поля:**
- **Заголовок:** если не заполнен, создаётся автоматически (Задача #ID)
- **Описание:** детали задачи
- **Экспертная группа:** для CT-исследований (ПГШ, Молочная железа, и т.д.)
- **Aria доступность:** Новый / Уже есть

**Поля для CT:**
- **Корпус для CT:** где проводится исследование
- **Аппарат CT:** название оборудования
- **Корпус для лечения:** где планируется лечение
- **Аппарат для лечения:** оборудование для лечения
- **Контроль по дыханию:** тип контроля дыхания

3. Прикрепите файлы при необходимости
4. Нажмите **«Создать задачу»**

### Просмотр своих задач

1. На главной странице отображаются **только ваши задачи**
2. Задачи других врачей вам не видны

### Статусы задач

| Статус | Описание |
|--------|----------|
| 🟨 **Ожидает** | Задача создана, но ещё не обрабатывается |
| 🟦 **В работе** | Физик обрабатывает задачу |
| 🟩 **Выполнена** | Задача завершена |
| 🟥 **Отменена** | Задача отменена |

**Важно:** Врач не может менять статус задачи. Это могут делать только физики и администраторы.

### Общение с физиком

1. Найдите задачу в списке
2. Нажмите на номер задачи или иконку чата
3. Откроется страница чата
4. Введите сообщение в поле ввода
5. Нажмите **«Отправить»** или клавишу Enter

**Особенности:**
- Вы видите только чаты своих задач
- Сообщения от физиков отображаются в реальном времени
- Индикатор «печатает» показывает, когда физик набирает сообщение

### Непрочитанные сообщения

- Красный значок в верхней панели показывает количество непрочитанных сообщений
- Нажмите на иконку **«Задачи»** для просмотра списка задач с непрочитанными сообщениями
- После прочтения сообщения значок исчезает

## Для физиков

### Просмотр всех задач

1. На главной странице отображаются **все задачи системы**
2. Вы видите задачи всех врачей

### Изменение статуса задачи

**Только физики и администраторы могут менять статус задачи.**

1. Найдите задачу в списке
2. Откройте задачу (нажмите на номер)
3. В правой панели найдите поле **«Статус»**
4. Выберите новый статус:
   - **Ожидает** — задача ожидает обработки
   - **В работе** — вы начали обработку
   - **Выполнена** — задача завершена
   - **Отменена** — задача отменена
5. Статус сохранится автоматически

### Добавление комментария

1. Откройте задачу
2. Перейдите на вкладку **«Чат»**
3. Введите сообщение
4. Нажмите **«Отправить»**

### Поиск задачи

Используйте фильтрацию на главной странице:
- По статусу
- По типу исследования
- По врачу

## Для администраторов

### Доступ к админ-панели

1. В верхней панели нажмите на иконку **«Админ»**
2. Откроется главная страница админ-панели

### Статистика системы

На главной странице админ-панели отображается:
- **Всего пользователей** — с разбивкой по ролям
- **Всего задач** — с разбивкой по статусам
- **Чаты задач** — активные за последние 7 дней
- **Личные чаты** — количество и активность
- **Файлов** — общее количество и размер
- **Файлы по типам** — изображения, PDF, DICOM, архивы

### Управление пользователями

#### Регистрация пользователя

1. Перейдите в **Админ → Пользователи**
2. Нажмите кнопку **«Добавить пользователя»**
3. Заполните форму:
   - **Имя пользователя:** уникальный логин
   - **Пароль:** временный пароль (минимум 3 символа)
   - **Роль:** Врач или Физик
   - **Фамилия, Имя, Отчество:** данные пользователя
   - **Отделение:** выберите из списка
4. Нажмите **«Зарегистрировать»**

#### Отделения по умолчанию

**Для врачей:**
- РО1, РО2, РО3, РО4, РО5

**Для физиков:**
- ФРО

**Для администраторов:**
- ТП

#### Смена пароля пользователя

1. Найдите пользователя в списке
2. Нажмите на иконку ключа 🔑
3. Введите новый пароль
4. Нажмите **«Изменить пароль»**

Сообщите новый пароль пользователю через защищённый канал связи.

#### Удаление пользователя

1. Найдите пользователя в списке
2. Нажмите на иконку корзины 🗑️
3. Подтвердите удаление

⚠️ **Внимание:** Удаление пользователя приведёт к потере связи с его задачами и сообщениями.

### Управление задачами

#### Просмотр всех задач

1. Перейдите в **Админ → Задачи**
2. Отобразятся все задачи системы

#### Редактирование задачи

1. Найдите задачу в списке
2. Нажмите на иконку карандаша ✏️
3. Измените поля:
   - Заголовок
   - Описание
   - Статус
   - Приоритет
   - Врач (исполнитель)
   - Детали исследования
4. Нажмите **«Сохранить»**

#### Удаление задачи

1. Найдите задачу в списке
2. Нажмите на иконку корзины 🗑️
3. Подтвердите удаление

⚠️ **Внимание:** Удаление задачи приведёт к потере всех сообщений и файлов.

#### Массовые операции

**Массовое изменение статуса:**
1. Отметьте задачи галочками
2. В поле **«Действие»** выберите **«Изменить статус»**
3. Выберите новый статус
4. Нажмите **«Выполнить»**

**Массовое изменение врача:**
1. Отметьте задачи галочками
2. В поле **«Действие»** выберите **«Изменить врача»**
3. Выберите нового врача
4. Нажмите **«Выполнить»**

**Массовое удаление задач:**
1. Отметьте задачи галочками
2. В поле **«Действие»** выберите **«Удалить задачи»**
3. Подтвердите удаление
4. Нажмите **«Выполнить»**

## Личные чаты

### Начало личного чата

1. Нажмите на иконку **«Чаты»** в верхней панели
2. Нажмите кнопку **«Новый чат»**
3. Выберите собеседника из списка
4. Нажмите **«Начать чат»**

### Отправка сообщения

1. Откройте чат
2. Введите сообщение в поле ввода
3. Нажмите **«Отправить»** или клавишу Enter

### Прикрепление файлов к сообщению

1. В окне чата нажмите на иконку скрепки 📎
2. Выберите файл
3. Файл прикрепится к сообщению
4. Отправьте сообщение

### Статус «Печатает»

Когда собеседник набирает сообщение, вы увидите индикатор **«Печатает...»** под полем ввода.

### Непрочитанные личные сообщения

- Красный значок рядом с иконкой **«Чаты»** показывает количество непрочитанных сообщений
- После прочтения сообщения значок исчезает

## Работа с файлами

### Поддерживаемые форматы

| Тип | Форматы |
|-----|---------|
| Изображения | PNG, JPG, JPEG, GIF, BMP, WEBP |
| Документы | PDF, DOC, DOCX, XLS, XLSX, TXT |
| DICOM | DCM, DICOM |
| Архивы | ZIP, RAR, 7Z |

**Максимальный размер файла:** 50 MB

### Загрузка файлов в задачу

**Врач:** может загружать файлы только в свои задачи

**Физик и Админ:** могут загружать файлы в любые задачи

1. Откройте задачу
2. Перейдите на вкладку **«Файлы»**
3. Нажмите **«Добавить файлы»**
4. Выберите один или несколько файлов
5. Нажмите **«Загрузить»**

### Просмотр файлов

- **Изображения:** нажмите для просмотра в полном размере
- **PDF:** открывается в браузере
- **Остальные файлы:** скачиваются на компьютер

### Скачивание файлов

1. Найдите файл в списке
2. Нажмите на иконку файла или название
3. Файл скачается на компьютер

### Удаление файлов

**Врач:** может удалять только свои файлы

**Физик:** не может удалять файлы

**Админ:** может удалять любые файлы

1. Найдите файл в списке
2. Нажмите на иконку корзины 🗑️
3. Подтвердите удаление

## Пользователи онлайн

1. Нажмите на иконку **«Онлайн»** в верхней панели
2. Отобразятся пользователи, сгруппированные по отделениям
3. Для каждого пользователя указано:
   - Фамилия И.О.
   - Роль
   - Время последнего появления в сети

**Онлайн-статус:**
- Пользователь считается онлайн, если он был в сети менее 5 минут назад

## Частые вопросы

### Как сменить пароль?

Обратитесь к администратору системы. Администратор может сменить пароль в разделе **Админ → Пользователи**.

### Как найти задачу?

Используйте фильтрацию на главной странице:
- По статусу
- По типу исследования
- По врачу

### Что делать, если забыл пароль?

Обратитесь к администратору системы для сброса пароля.

### Как прикрепить большой файл?

Максимальный размер файла — 50 MB. Если файл больше:
- Сожмите файл в архив
- Разделите на части
- Используйте облачное хранилище и отправьте ссылку в чате

### Почему не загружается файл?

Возможные причины:
- Файл слишком большой (> 50 MB)
- Неподдерживаемый формат
- Проблемы с подключением к интернету

### Как узнать, кто онлайн?

1. Нажмите на иконку **«Онлайн»** в верхней панели
2. Увидите список пользователей по отделениям

### Можно ли редактировать задачу?

**Врач:** не может редактировать задачу после создания

**Физик:** может изменить только статус задачи

**Администратор:** может редактировать все поля задачи

### Как отменить задачу?

Изменение статуса на «Отменена» доступно физикам и администраторам. Врач может попросить отменить задачу через чат.
